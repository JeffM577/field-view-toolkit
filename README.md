# Field View Toolkit

Reusable frontend patterns for field operations and roofing workflows.

## Goal

Create clean, practical UI patterns for teams managing projects, material requests, documents, schedules, and field updates.

## Planned Features

- React dashboard shell
- Ant Design layout and navigation
- Light/dark theme tokens
- Reusable loading, empty, and error states
- Data-table examples for operational workflows
- Document workflow examples

## Tech Stack

- React
- React Router
- React Query
- Ant Design
- Tailwind CSS
